Aesthetically Pleasing Tetris

React/Webpack
Flask, SocketIO
NGINX, Gunicorn
A tetris website I made for 2017 WebHacks. The most difficult part was by far creating the backend for the multiplayer mode, which relies heavily on sockets. There were 2 serious hurdles in the development process: creating a pairing system, and actually using that system to send the players their opponents' moves. I ended up using a global list as a queue, and random IDs that the server generates when a new user enters the loading page.
